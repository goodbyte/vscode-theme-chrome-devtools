# Change Log

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.5] - 03-01-2022
### Added
- New color for "import", "from" and "module" keywords.
### Changed
- Increased yellow brightness.
- Toned down foreground color.
### Fixed
- RegExp consistency.
### Removed
- Variable color inside template strings.
- Object literal key color.

## [1.0.4] - 03-01-2022
### Fixed
- A TextMate scope causing trouble.

## [1.0.3] - 03-01-2022
### Changed
- Minor changes, fixes and cleaning.
- Decided not to follow the color scheme to the letter, in favor of more consistency between languages.

## [1.0.2] - 28-12-2021
### Changed
- Tweaked the colors a little bit for better readability, and dropped the background color.

## [1.0.1] - 27-12-2021
### Changed
- Marketplace logo.

## [1.0.0] - 27-12-2021
- Initial release.
